
//Superstore Sales Analysis

Projeto de análise exploratória de dados utilizando a base fictícia Sample Superstore, com foco na identificação de padrões de vendas, lucratividade e desempenho comercial através de visualizações e dashboards interativos.

//Objetivo

O objetivo deste projeto é aplicar conceitos de:

- Limpeza de dados
- Análise exploratória (EDA)
- Visualização de dados
- Geração de insights de negócio
- Construção de dashboards

utilizando Python e bibliotecas voltadas para Data Analytics.

//Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

//Dataset

O conjunto de dados utilizado é o famoso Sample Superstore Dataset, amplamente utilizado para estudos de análise de dados e Business Intelligence.

O dataset contém informações como:

- Pedidos
- Categorias de produtos
- Regiões
- Clientes
- Vendas
- Lucro
- Descontos

//Análises Realizadas

- Análise de vendas por região
- Análise de lucro por categoria
- Identificação de produtos mais rentáveis
- Distribuição de vendas
- Correlação entre variáveis
- Visualizações gráficas para apoio à tomada de decisão

//Principais Insights

- Identificação das regiões com melhor desempenho em vendas.
- Comparação entre faturamento e lucratividade.
- Categorias que apresentam maior retorno financeiro.
- Tendências observadas no comportamento das vendas.

OBS: Os insights serão expandidos conforme o desenvolvimento do projeto.

//Como Executar

Clone o repositório:

git clone https://github.com/skull-np/superstore_analysis.git

Entre na pasta do projeto:

cd superstore_analysis

Instale as dependências:

pip install -r requirements.txt

Execute o notebook:

jupyter notebook
